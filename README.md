# Data Transformation Common Pattern DBT

A dbt (data build tool) project for data transformation using DuckDB as the database backend.

## Prerequisites

- Python 3.8+
- dbt-core
- dbt-duckdb adapter
- Task (Taskfile executor)

## Installation

```bash
# Install dbt with DuckDB adapter
pip install dbt-core dbt-duckdb

# Install Task (macOS)
brew install go-task/tap/go-task
